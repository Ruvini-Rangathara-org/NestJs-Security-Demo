  <h1 align="center">Security Demo Application with NestJS</h1>
 
## Description

This application is built with NestJS and provides a simple authentication mechanism using JWT (JSON Web Tokens).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Install the following packages
````
npm install --save @nestjs/jwt
````
